# Dados do projeto

Coloque o dataset Tracebase neste projeto com a seguinte estrutura:

```text
tracebase/
  complete/
  incomplete/
  synthetic/
  australia/
```

Nesta etapa do projeto, use apenas:

```text
tracebase/complete/
```

O objetivo inicial e transformar cada arquivo de medicao em uma linha com
caracteristicas estatisticas de consumo, usando o tipo de eletrodomestico
como target (`y`).
